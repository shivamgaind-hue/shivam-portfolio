Capstone Project

AI-Based Insect Detection System

Built an insect detection device inspired by a Malaise trap, using controlled lighting, edge computing, and YOLOv8 models for insect classification.

  • Designed a trap workflow where insects enter a photo chamber for automated imaging.
  • Used ESP32-controlled LED panels, Jetson Nano processing, and two YOLOv8 classifiers.
  • Validated model focus with Grad-CAM heatmaps and confidence-score outputs.
  • Research is in progress toward an official publication, with demos presented to engineering leadership at Penn State and Ohio State.
